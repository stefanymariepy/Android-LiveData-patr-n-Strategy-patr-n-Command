# 📂 Implementación de Patrones de Diseño

Este repositorio contiene la implementación práctica de tres patrones de diseño de comportamiento, aplicados a casos reales en el desarrollo de software, tal como se utiliza en frameworks, librerías de inteligencia artificial y videojuegos.

Cada ejemplo está **simplificado, comentado y documentado** para su fácil comprensión y ejecución.
